Структура базы данных



users              — id, name, created_at

forum_categories   — id, icon, name

place_categories   — id, icon, name

places             — id, name, address, city, country, description,

                     working_hours, price_range, category_id

reviews            — id, user_id, place_id, title, body, rating,

                     category_id, country, city, created_at

review_photos      — id, review_id, file_path



Связи:

•  reviews.user_id → users.id

•  reviews.place_id → places.id 

•  reviews.category_id → forum_categories.id

•  places.category_id → place_categories.id

•  review_photos.review_id → reviews.id
